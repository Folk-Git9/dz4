# Калькулятор на Python

## Описание проекта

Терминальная программа-калькулятор на языке Python, поддерживающая базовые математические операции:

- Сложение (`+`)
- Вычитание (`-`)
- Умножение (`*`)
- Деление (`/`)
- Возведение в степень (`^`)
- Квадратный корень (`√`)

Все функции реализованы в **параметрическом виде**, что позволяет легко использовать их в других проектах и проводить модульное тестирование.

---

## Файлы проекта

- `calculator.py` — основной файл программы с реализацией всех функций.
- `tests/` — папка с модульными тестами:
  - `test_calculator.py` — тесты для всех функций калькулятора.
- `README.md` — техническая документация проекта.
